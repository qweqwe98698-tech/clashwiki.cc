### 1. IP 被污染或列入黑名单

OpenAI 会检测请求来源的 IP 是否属于“干净”的原生 IP。如果你用的机场节点是共享 IP 或机房 IP（如 AWS、Google Cloud 等），很容易被标记。

### 2. 地区限制

ChatGPT 目前仅对部分国家和地区开放。如果你的节点 IP 归属地不在支持列表，就会直接拒绝访问。

### 3. 过度使用或异常行为

同一 IP 下大量请求、频繁切换账号等，也会触发风控。

- 使用 **纯净原生 IP** 的节点（家庭宽带 IP 或住宅 IP 最佳）
