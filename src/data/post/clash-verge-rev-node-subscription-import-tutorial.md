## 二、节点订阅导入图文极简教程

### 第一步：下载并安装Clash Verge Rev
- 官方GitHub Release页面下载对应系统版本。
- Windows用户选择.exe安装包，Mac用户选择.dmg文件。

### 第二步：获取机场订阅链接
- 以[👉守候网络机场]()为例，注册登录后，在用户中心找到“一键订阅”按钮，复制Clash/Meta专用订阅链接。
- 注意：不同机场的订阅链接格式类似，都是http或https开头的一串字符。

### 第三步：导入订阅到Clash Verge Rev
1. 打开Clash Verge Rev，点击左侧“订阅”选项卡。
2. 点击右上角的“+新建”按钮。
3. 在弹出的窗口中，粘贴你复制的订阅链接。
4. 点击“确定”，软件会自动拉取节点列表。
5. 回到“节点”选项卡，你会看到所有可用节点，选择一个延迟最低的即可。

### 第四步：开启代理
- 点击右下角的“系统代理”开关，开启后状态变为绿色。
- 现在你可以打开浏览器访问Google、YouTube等网站验证是否成功。

> 小贴士：如果节点延迟高或无法连接，可以尝试切换其他节点，或者更新订阅。
