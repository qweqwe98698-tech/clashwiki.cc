## 三、Clash Verge Rev 订阅导入图文教程（Windows / Mac 通用）

以下教程基于最新版 Clash Verge Rev v1.3.8，界面简洁，操作简单。

### 第一步：下载并安装 Clash Verge Rev

- **Windows用户**：前往 [GitHub Releases](https://github.com/clash-verge-rev/clash-verge-rev/releases) 下载 `.exe` 安装包，双击安装。
- **Mac用户**：下载 `.dmg` 文件，拖入Applications文件夹。

安装后打开软件，界面如下：

![Clash Verge Rev主界面](https://via.placeholder.com/600x400?text=Clash+Verge+Rev+Main+UI)

### 第二步：复制机场订阅链接

登录你的机场后台（以老猫云机场为例），找到“订阅”或“我的订阅”页面，点击“复制订阅链接”。

![老猫云机场订阅链接](https://via.placeholder.com/600x400?text=老猫云订阅链接)

### 第三步：在Clash Verge Rev中导入订阅

1. 点击左侧菜单栏的“订阅”图标。
2. 点击“添加”按钮。
3. 在弹出的窗口中，填写：
   - **名称**：任意（如“老猫云”）
   - **URL**：粘贴刚才复制的订阅链接
4. 点击“导入”，稍等几秒，节点列表就会出现在下方。

![导入订阅步骤](https://via.placeholder.com/600x400?text=导入订阅步骤)

### 第四步：启用代理并选择节点

1. 回到主界面，点击“代理”图标。
2. 选择一个节点（推荐“香港-专线”或“日本-专线”）。
3. 点击顶部开关，开启代理。

![选择节点](https://via.placeholder.com/600x400?text=选择节点)

恭喜！你已经成功连接。打开浏览器试试访问Google或YouTube吧。

如果遇到问题，可以检查：
- 确保系统代理已开启（在Clash Verge Rev的“设置”中勾选“系统代理”）。
- 尝试切换不同的节点。
- 更新订阅（右键点击订阅 -> 更新）。
