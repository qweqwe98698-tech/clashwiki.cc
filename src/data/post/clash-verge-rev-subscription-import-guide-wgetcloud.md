## 一、Clash Verge Rev 节点订阅导入图文教程

### 1. 下载并安装 Clash Verge Rev
- 前往 GitHub Releases 页面下载对应系统版本（Windows 选 .exe，Mac 选 .dmg）。
- 安装后打开，界面简洁，默认开启核心。

### 2. 获取订阅链接
- 在机场后台（比如 WgetCloud）复制你的 Clash 订阅链接。
- 注意：部分机场提供的是 Base64 或 SSR 链接，但 Clash Verge Rev 支持直接导入 Clash 格式。

### 3. 导入订阅
- 点击左侧“订阅”图标，再点击“添加”。
- 粘贴订阅链接，点击“确认”。
- 软件自动解析，显示可用节点列表。
- 右键点击订阅，选择“更新”确保节点最新。

### 4. 启动代理
- 在“代理”页面选择节点模式（推荐 Rule 或 Global）。
- 点击“启用系统代理”，即可科学上网。

> 小技巧：如果节点无法连接，尝试切换节点或更新订阅。WgetCloud 的节点延迟普遍在 100ms 以内，稳定性很好。
